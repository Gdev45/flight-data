# Flight Data CLI

A simple terminal tool to fetch and display flight data. Works on Linux, macOS, and Windows.

Get live flight information right from your terminal with an easy-to-use CLI.

---

## Features

- Fetch flight data from public APIs or local sources  
- Runs directly in the terminal  
- Cross-platform: Linux, macOS, Windows  
- Easy to install and run using Python

---

## Installation

### Recommended: Using pipx (isolated CLI)

```bash
pipx install git+https://github.com/Gdev45/flight-data.git

```
## Run using

```bash
flight-data
```
